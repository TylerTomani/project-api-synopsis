Part 1 Setup - API

Go to,
https://console.cloud.google.com/cloud-resource-manager
Click on Create Project on the Manage Resources pages,
(or)
Click Dashboard and go to to Select A Project

Name the project and click Create

Go to API's & Services → Enable API's & services,
Then,
Search for and enable the YouTube Data API v3

Click on Create Credentials,

for this api, choose

  • The YouTube Data v3 api
  • & Public Data
  • Click Done

now we have an api key, go to Credentials to copy it

Paste the api key into a variable in your script
Remember delete when not using unless encrypted,restricted and secure
